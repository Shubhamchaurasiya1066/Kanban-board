# 🚀 Kanban Task Board (Trello Style)

A modern **Kanban Task Management Board** built using **React + Tailwind CSS**.
This project helps users manage tasks in different stages like **To Do, In Progress, and Done**.

---

## ✨ Features

* ➕ Add Task with Priority (High, Medium, Low)
* 📝 Edit Task (Click to Edit)
* ❌ Delete Task
* 🔄 Move Task between columns
* 🎨 Priority-based colored borders
* 💾 Data saved in LocalStorage (persists after refresh)
* 📱 Responsive UI (Mobile + Desktop)
* 🎯 Clean UI using Tailwind CSS

---

## 🧩 Tech Stack

* React (Vite)
* Tailwind CSS
* JavaScript (ES6)
* LocalStorage
